# Task Manager App

## Features
- Add, update, delete tasks
- Priority levels (low, medium, high)
- Due dates
- Filter (completed/pending)

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
